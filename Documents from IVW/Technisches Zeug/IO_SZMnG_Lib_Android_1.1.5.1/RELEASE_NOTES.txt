RELEASE NOTES INFONLINE SZMNG LIBRARY ANDROID
=============================================

INFOnline SZMnG Library for Android v1.1.5.1
Build: 08e0bc566809b04bc5e02a337603790817bb74b9
Release Date: 08.12.2016

CONTENTS OF THIS PACKAGE
========================

This release of INFOnline SZMnG Library for Android consists of the following:

infonlinelib_1.1.5.1.aar
	The IO SZMnG Lib Android v1.1.5.1 itself as a binary distribution file

INFOnlineLibSample_AndroidStudio
	A sample application for Android Studio to demonstrate the integration and usage if the INFOnline SZMnG Lib.

CHANGELOG.txt
   	Version history / change log

RELEASE_NOTES.txt
   	What you're looking at right now.


REQUIREMENTS
============

Development:
- Android SDK Release 24 and higher
- Android Studio 2.0 and above
- Google Play Services 8.4 and higher

Operation:
- Android OS v2.3.3 (API Level 10) and higher


INTEGRATION/USAGE
=================

For a detailed instructions, please refer to the corresponding Integration Guide IOLib SZMnG for Android.
